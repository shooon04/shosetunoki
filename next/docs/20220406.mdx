# そもそもWordって？
「Word」とはMicrosoftが提供するソフトウェアのことです。
 
ExcelやPowerPointと並んで「Office製品」と呼ばれることもあります。
学校や職場で使ったことや聞いたことがある方も多いのではないでしょうか？
 
Wordは特に文書の作成に特化したソフトウェアで、横書きはもちろん縦書きもできてしまいます。
慣れれば自作の本のレイアウトを組むこともできます。
 
小説の執筆だけでなく、他の用途にも使いやすいことが魅力ですね。
 
Wordには無料版と買い切り版があるため、「ちょっと試してみたいな」という方はまず無料版をダウンロードしてみることをオススメします。
 
 
# Wordのメリット・デメリットは？
まずはWordのメリットとデメリットを整理してみましょう。
 
## メリット
・校正機能

<amp-img src="20220406-1.jpg" alt="ワード校正機能" width="700" height="373" layout="responsive" />

誤字・脱字、表記揺れや妙な言い回しの箇所を波線で示してくれます。
 
校正ソフトは数多くありますが、多くは執筆ソフトとは別の、独立したソフトです。
 
誤字や表記揺れを調べたい箇所を選択＆コピーして、校正ソフトへとペーストする。
 
この一連の作業を何度も何度もくり返すと、さすがに面倒になってくるのではないでしょうか。
 
もちろん校正専門のソフトと比べるとWordの校正機能は少なく、確実で正確な校正が行えるかどうかと言われると首を傾げざるをえません。
 
しかしWordは、執筆をしながらリアルタイムで校正が行えるという利点があります。
 
執筆中はできる限り集中力を保っていたいですし、何より余計な手間はかけたくない。そんな時に活用していきたいですね。
 
 
・変更箇所の記録

<amp-img src="20220406-2.jpg" alt="ワード変更履歴の例" width="700" height="373" layout="responsive" />

ある文章を修正したけど、前の文章の方がよかったかもしれない。
でもデータ消しちゃった……。
 
こんな時に便利なのが、「変更箇所の記録」機能です。
 
前のデータを保ちながら新しい文章を書くことができるため、紙面上での推敲と同じような役目を果たしてくれます。
 
筆者のような貧乏性や物忘れがひどい方にもオススメの機能です。
 
・読み上げ機能
入力したテキストを合成音声で読み上げてくれる機能です。
 
この機能はOffice2010バージョン以降から実装されていて、目では発見できなかった誤字・脱字や文章の違和感の発見に役立ってくれます。
 
もちろん、他の読み上げソフトでも代用は可能ですが、「読み上げたい文章をソフトに入力する」という工程を挟むため、手間がかかります。
 
Wordなら他のソフトを立ち上げることなく、好きな場所から何度でも読み上げてもらえるため手間がかからず、また集中力も途切れません。
 
読み上げ音声も変えられるため、好きな音声に変えて楽しんでもよいかもしれませんね。
 
・レイアウトができる
「レイアウト」とは、文章をどこに配置するか、どこに挿絵を入れるか、などの編集をすることです。
 
レイアウトには行数と列数の設定も含まれていて、これが特に関係してくるのが文学賞への応募です。
 
文学賞へ応募する際、応募規定の中に30×40行、40×40行、400字詰めなどの指定を見たことがある方もいるかと思います。
 
また、データで応募する際にWordに限る文学賞がちらほらと存在します（ジャンプホラー小説大賞、ポプラ社小説新人賞など）。
 
Webへの投稿のほかに公募を目指している方にとっては、Wordを導入した方が楽になる場面が多く出てくる可能性があります。
 
余談ではありますが、「自作の小説を自力で書籍化・電子書籍化しよう」となった際にもWordがあると便利です（実際に筆者もWordを使用して本を作りました）。
 
## デメリット
・初期費用がかかる
一番のデメリットは、やはりお金がかかってしまうことです。
 
定額課金版、買い切り版など、支払いの形態はさまざまですが、多少なりともお金がかかってしまいます。
 
もともとパソコンに入っているのであれば気にする必要はありませんが、Office製品が入っていないパソコンを使用している方は財布が痛んでしまいます。
 
なお、無料版もMicrosoftから発表されてはいますが、こちらはインターネット環境下でないと使用できないというデメリットがあります。
 
オフライン環境下でも作業したい、という方には無料版はオススメできません。
 
・編集ソフトが限られる
Wordを使用して作ったデータを保存する際、「docx」というファイル形式で保存されます。
 
ファイル形式とは、データの種類のことです。
jpgやpng、txtなど、見かけたことのある人も多いのではないでしょうか。
 
データのファイル形式とソフトは、言うなれば鍵と錠前のような関係です。
 
ファイル形式を開くことのできるソフトがなければ、データの閲覧はおろか編集すら不可能なのです。
 
これの何がデメリットなのか。いまいちピンと来ない方もいると思います。
 
例を挙げてみましょう。
 
①Wordで書いた小説データを記録媒体に保存して、友人に見せようとした。
②待ち合わせ場所へ向かう道中、より最適な言い回しを見つけて、たまたま目についたネットカフェに置いてあるパソコンで編集しようとした。
③だけどネットカフェのパソコンにWordが入っていなかった。
 
この場合、Wordで書いた小説データ（錠前）を開くためのソフト（鍵）がないため、編集することはできません。
 
また、docx形式の編集はGoogleドキュメントでも可能ですが、Googleにログインする手間があります。
 
そのほかにも、Word以外のソフトでデータを閲覧するために配置がずれてしまったり、思い通りの文書ではなくなってしまうかもしれません。
 
文書の編集をする際にWordしか推奨されない、という点はデメリットと言えるでしょう。
 
 
# 執筆にWordは必要？
結論から言うと、必要ありません。
 
先程「Wordのメリット・デメリットは？」の項目を読んでいただきました。
 
一見するとメリットの方が多く、Wordの方がよいと思われる方もいると思います。
 
しかしメリットには一つの共通点があります。
 
それは「代用が可能であること」です。
 
この記事で挙げたWordを使用するメリットの大半は、校正機能や読み上げ機能など、他のソフトで代用が可能なものです。
 
校正機能は校正ソフトが無料で存在しますし、読み上げ機能もまた同様です。
 
レイアウトに関しては公募を目指す方に限った需要になりますし、「自力で書籍化・電子書籍化したい」という方向けのソフトとしては、Wordよりも直観的で使い勝手のよいInDesign（Adobe社）が挙げられます。
 
他のソフトを利用することで手間は増えますが、Wordのメリットの機能は唯一無二というほどではありません。
 
メリットのためにお金を払うかと言われると迷ってしまう方も多いのではないでしょうか。
 
一方、デメリットであるお金がかかること、編集ソフトが限られること。
この二点は何物にも代えられません。
 
どうしても避けたいのであれば、他のソフトで執筆してもよいと思います。
 
Wordがなくても小説は書けます。
 
ただし賞に応募する際、どうしてもWordのデータが必要になる場合は、その限りではありません。応募規定をチェックして導入を検討してみてくださいね。
 
 
# おまけ　スマホ版のWordってあるの？
iOS（iPhone）版、Android版ともにあります。
 
スマホ版のWordには無料版・サブスクリプション版が用意されているため、パソコンは持っていないけどWordは使ってみたい、という方でも使用できます。
 
ただし無料版では、上で紹介した「変更箇所の記録」「校正機能」は利用できませんので注意が必要です。
 
 
# まとめ
ここまでWordを使用するメリット・デメリットをもとに、「小説の執筆にWordは必要か」という点について考えてきました。
 
これまでに挙げたメリットとデメリットをもう一度確認してみましょう。
 
Wordを使用するメリット
- 校正機能
- 変更履歴を残すことができる
- 読み上げ機能がついている
- レイアウトができる
 
Wordを使用するデメリット
- 初期費用がかかる（※無料版もあり）
- 編集ソフトが限られる
 
ただしほとんどの「メリット」には代替ソフトが存在します。
 
執筆にWordが必要とは限らない、Wordがなくても執筆は可能、という結論に至りました。
 
もちろんWordは便利なソフトですし、筆者も生粋のWordユーザーです。しかし、お金のことや出先での編集が困難などのデメリットに悩んだこともあります。
