# 紙の辞書をペラペラめくってみよう

## 紙の辞書のメリット

紙辞書のメリットは自分の知らなかった言葉を知れるという点です。

電子辞書は言葉を入れないと意味が出てきません。ということはその言葉を知らなければ、まず調べることすらもできないのです。スタートラインに立つ前にはじき返されてしまうのです。

紙の辞書は、適当に開いてみるとたくさんの言葉の意味が載っています。そして大体の場合、類義語や対義語が載っています。
例えば、「縦断」という文字を見ると対義語で「横断」と書かれています。また、最後には類義語として「縦貫」が載っています。

僕は辞書に載っている類義語を使ったり、辞書の後ろに大体載っている「付録」などを活用して書いています。実際に今、この記事を書いているときにも隣に紙の辞書を置いています。

## めくるだけでも効果はある！？

ちょっと暇だなと思った時はTwitterなどのネットを見るのもいいですが、辞書を読んでみませんか？
適当にページを開いてそのページに目を通すのです。大体の場合、収穫があります。聞いたことはあったけど意味がいまいちわからない言葉や、難しい漢字を知ることができます。そして知ったことを小説に反映するのです。

紙の辞書というのは重いし、かさばりますが、語彙を知ると言う点で一番使えます。

# ネットを活用？！

## ネットのメリット

ネットを使うメリットもあります。例えば、紙の辞書に載っていないことを調べる時などです。紙の辞書に載っていないことなんてある？と思われるかもしれませんが、専門的なことを調べる場合はネットを活用したほうがいいことがあります。

僕の場合は、花言葉をテーマにした作品を書いているので花言葉を調べるなど、活用方法は無限です。

## ネットを使う時に注意して欲しいこと

しかし、ネットを使うときに注意しなければいけない点が二つあります。

一つ目は、参考文献としてページのURLを最後に貼るか、何を使ったかを自分がすぐにわかるようにしておくことです。

二つ目は、複数のサイトを見ることです。
一つのサイトだけをみて、自分に取り込むのではなく、複数のサイトを見た上で、情報を精査し作品に生かしましょう。

# 他人の小説からヒントを得る

## 自分の書くジャンル以外の本は読まないは間違い！

あなたは一つのジャンルに偏って本を読んでいませんか？
異世界転生だったら異世界ファンタジーの作品、ミステリーだったらミステリーなど、自分の書きたい作品のジャンルの本だけを読んでいませんか？

それは間違いです！

一つのジャンルだけ読んでも表現の幅は広がりません！

たとえあなたが一つのジャンルに絞って小説を書くとしても、他ジャンルの本は読むべきです。もう少し言うと、複数の作家さんの作品を読むと良いです。作家さんによって表現の方法は十人十色です。絶対に読むべきです。

## おすすめの小説サイト

紙の本ももちろんいいですが、お金がかかります。お金をかけずに学ぶのに一番いい方法は、「小説サイトで読む」です。ブックマークをつければすぐに見返すことができるので効率的です。
僕のおすすめは「小説家になろう」や「カクヨム」で読むことです。

# まとめ

今回の記事では、「ボキャブラリーの広げ方」について僕の実体験を交えながら書かせていただきました。

- 辞書を活用する
- 専門的なことはネットを使う
- 他人の小説を読む

ことが大切です。やることいっぱいで面倒くさいなと思うかもしれません。しかし、語彙力を広げるには、まず知ることが第一です。
