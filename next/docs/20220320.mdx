# 起承転結

まず、そもそも起承転結とは何かを解説します。

## 一般的な起承転結とは

起承転結は、コトバンクでは下記のように定義されています。

> 漢詩、とくに絶句（4句の詩）の構成法の用語。「起承転合」ともいう。第一句を起句、第二句を承句、第三句を転句、第四句を結句という。起句はうたい起こしで、一首の意を提起するものであるから、高い風格、非凡な着想が必要である。承句は起句を受けて、詩意を発展させる。転句は場面を転換するが、人の意表に出るような奇抜さが必要であり、一つの見どころとなる。結句は転句を受けつつ、全体を収束し、余韻を言外に漂わす。

引用元：コトバンク（ https://kotobank.jp/word/%E8%B5%B7%E6%89%BF%E8%BB%A2%E7%B5%90-12404 ）

つまり、下記のように絶句の一つの雛形とも呼べる構成法です。

- 起句：物語を起こし、世界観を伝える句
- 承句：起句を承けて、内容を広げる句
- 転句：内容を転換させ、見どころを作る句
- 結句：物語を結び、余韻を残す句

## 小説の起承転結とは

絶句の起承転結を小説の世界に落とし込むと、下記のようになります。正直、ほとんど意味は変わりません。

- 起：物語を起こし、世界観を伝える
- 承：物語の起こりを承けて、内容を広げる
- 転：内容を転換させ、見どころを作る
- 結：物語を結び、余韻を残す

四コマ漫画をイメージすると、創作における起承転結がどのようなものかわかりやすい思います。（と言いつつ、四コマ目に転を持ってくる四コマ漫画もありますが）

# 起承転結の例

次に、起承転結の具体的な例を「桃太郎」を使って説明します。

## 起の例

桃太郎の起の文は、「桃から生まれた男の子に、桃太郎と言う名をつけた」です。

物語の起こりとして、主人公である桃太郎にどんな特徴があるかを描きます。
ここで、桃太郎は桃から生まれるという周囲の人間とは違う生まれ方をした、言わば特別な人間であることを描き、この物語が桃太郎を中心に広がっていくことを読者と共有します。

## 承の例

桃太郎の承の文は、「悪い鬼の話を聞いた桃太郎は、鬼退治に出かけた」です。

物語の起こりを受け、桃太郎の住む世界と物語のゴールを描きます。
ここで、桃太郎の住む世界には、悪い鬼が存在し、桃太郎の目的が悪い鬼を退治しすることであることを描き、この物語のゴールが鬼退治であることを読者と共有します。
承の文では特に、桃太郎が周囲の子供より力のある人間として成長したり、犬猿雉を仲間にしたりと、物語に設定を付与することで、物語に厚みを持たせることができます。

## 転の例

桃太郎の転の文は、「桃太郎が鬼ヶ島で、鬼を倒した」です。

物語が転換し、桃太郎が鬼退治と言うゴールにたどり着いたことを描きます。
ここで、積み上げてきた伏線や設定を放出し、桃太郎が鬼と戦い、悪い鬼が蔓延る世界から鬼がいなくなった平和な世界へ転換したことを読者と共有します。
物語の一番盛り上がる部分でもあります。

## 結の例

桃太郎の結の文は、「桃太郎が宝物を持って、帰ってきた」です。

転換後の世界を描きます。物語の収束を描きます。
ここで、鬼たちがいなくなった世界がいかに平和な世界かを描き、一つの物語が終わったことを読者と共有します。

# 起承転結の小説への適用

最後に、起承転結をどのように小説へ適用すればいいかを解説します。

## 短編小説への適用

一話完結の短編小説においては、一話へのプロット作成に適用できます。
最初に起承転結を1～2文程度で表現し、物語の骨組みを作ります。そして、起承転結に設定を加筆していくことで、物語のプロットを作ることができます。
最初から本文を書いてしまうと物語の軸がぶれがちなので、プロットを活用し、本文を書いていくことをお勧めします。

## 長編小説への適用

長編小説への適用についても、短編小説への適用と考え方は同じです。
ただし、長編小説の場合は適用範囲が広がり、一話単位だけでなく、一章単位、物語全体へも適用できます。
物語全体の起承転結を作った後、物語全体の起の中で、さらに小さな起承転結を作り、これを繰り返すといったイメージです。
大きなプロットからどんどん小さなプロットを作っていくことで、物語の前半と後半で設定や軸がぶれたりすることを防ぎます。

## 適用は必須か

ここまで、短編小説と長編小説への適用方法を書きましたが、小説を書く際に起承転結を適用することは、必須ではありません。
あくまでも、一つの小説の構造として捉えてください。

そもそも起承転結は、絶句を作るために適した構造として生まれています。
現代の小説も、内容や展開によって起承転結が適する小説もあれば、適さない小説もあります。
例えば、SS小説の世界では、文字数が制限されて内容を広げることが難しく、「起転結」となる傾向があります。

参考：SS小説って何？　書き方も教えます！（ https://shosetunoki.com/media/20220317 ）

自分の書きたい小説は、起承転結という構成が向いているか否か、考えてから適用してみましょう。

# まとめ

いかがだったでしょうか。
今回の記事では、起承転結とその適用方法について説明しました。
小説を書き始める時に少しでも役立てば幸いです。

また、ぼくは小説の書き方は、自由で良いと考えています。
起承転結に沿ったかっちりとした小説も小説ですし、あえて起承転結を外した小説も小説です。
自分にとって書きやすい構成、書いてて楽しい構成を見つけ、楽しい執筆ライフを送りましょう。
