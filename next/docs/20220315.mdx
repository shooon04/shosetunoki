# 一般的な人称とは

人称は、コトバンクでは下記のように定義されています。


> 代名詞や動詞に現れる文法範疇(はんちゅう)の一種。次の三つの人称が区別される。（1）一人称。話し手「私」または話し手を含む人たち「私たち」。（2）二人称。聞き手「あなた」または聞き手を含む人たち（話し手を除いて）「あなた方」。（3）三人称。一人称、二人称以外の人や物。「彼」「彼女」「彼ら」「彼女ら」など。

引用元：コトバンク（ https://kotobank.jp/word/%E4%BA%BA%E7%A7%B0-110844 ）

つまり、下記のように話し手、聞き手、それ以外を区別するのが人称です。

- 一人称：話し手
- 二人称：聞き手
- 三人称：それ以外

## 人称視点とは

人称を小説の世界に落とし込むと、少しだけ意味が変わります。

- 一人称視点：私（主人公、または登場人物）の視点で物語が展開
- 二人称視点：あなた（主人公の前に居る誰か）の視点で物語が展開
- 三人称視点：天の声の視点で物語が展開

一人称視点は、「私が～」、「俺が～」という書き方で小説が展開されていきます。
一方で、三人称視点は、「Ａさんが～」、「Ｂさんが～」という書き方で小説が展開されていきます。
よく例えられるのが、小説の世界を覗くためのカメラを、誰が持っているかの違いです。

# 一人称視点の特徴

一人称視点は、主人公の視点で物語が進んでいき、下記三点の特徴があります。

- 状況を主人公の主観で書ける
- 主人公の心情を深堀りできる
- 主人公に共感しやすい

## 状況を主人公の主観で書ける

一人称視点では、状況を主人公の主観で書けます。

三人称視点では、作者と読者で状況を共有するために、状況の詳細な描写が必要となります。例えば、スライムという魔物に対し、どんな形状が、目・鼻・口はあるのか、色は何色か、というスライムの情報を一から描写する必要があります。

一方で、一人称視点では、作者と読者の持つ共通のイメージを、主人公の知識という形で表現できます。例えば、スライムという魔物に対し、「ドラゴンクエストで見たスライムそっくり」と描写すれば、作者と読者での共有ができます。
仮にドラゴンクエストのスライムを読者が知らなくとも、読者が検索すれば画像が出てきて、すぐに共有ができます。

## 主人公の心情を深堀りできる

一人称視点では、主人公に共感しやすいです。

三人称視点では、複数の登場人物の心情と行動を横断して物語が展開されるため、読者は特定の登場人物の心情にどっぷり浸かることは難しいです。

一方で、一人称視点では、常に主人公の心情と行動を追えばいいため、読者は自分を主人公に投影し、主人公の心情と行動に共感しやすくなります。

# 二人称視点の特徴

二人称視点は、あなたの視点で物語が進んでいき、下記三点の特徴があります。

- 状況を読者の主観で書ける
- 読者を巻き込んだ物語を書ける
- 読者に斬新さを与えられる

## 状況を読者の主観で書ける

二人称視点では、状況を読者の主観で書けます。

一人称小説と三人称小説では、物語の展開が登場人物のやり取りに限定されます。

一方で、二人称小説では、「あなたは目を覚ます」や「あなたはどう思いますか？」のように、読者を登場人物の一人として扱う描写ができます。

<h3>読者を巻き込んだ物語を書ける</h3>

二人称視点では、読者を巻き込んだ物語を書けます。

一人称小説と三人称小説では、読者は小説の世界を覗くことしかできず、登場人物に共感することで小説の世界に浸ります。

一方で、二人称小説では、読者も小説の世界の登場人物として扱うため、読者を小説の世界へより没頭させることができます。

## 読者に斬新さを与えられる

二人称視点では、読者に斬新さを与えられます。

一人称小説と三人称小説は、既にたくさんの小説が存在し、読者が何度も読んできた形式となります。

一方で、二人称小説は、小説の絶対数が少なく、読んだことがない読者も多いです。そのため、今まで読んだことがない斬新さを与えることができます。

# 三人称視点の特徴

三人称視点は、天の声の視点で物語が進んでいき、下記三点の特徴があります。

- 状況を客観で書ける
- 登場人物全員の心情を書ける
- 登場人物全員が知らないことを書ける

## 状況を客観で書ける

三人称視点では、状況を客観で書けます。

一人称視点では、全ての描写が主人公の視点となるため、状況の描写に主人公の心情というフィルターがかかったり、主人公の知識以上の描写ができません。例えば、作者と読者で状況を共有するために、状況の詳細な描写が必要となります。例えば、主人公の目の前で高度な戦術が展開されたとしても、主人公が戦術について詳しくなければ、説明の描写ができません。

一方で、三人称視点では、誰の心情のフィルターもかからず、主人公が知らない知識でも描写ができます。例えば、主人公の目の前で高度な戦術が展開され、主人公が戦術について詳しくなくとも、説明の描写ができます。

## 登場人物全員の心情を書ける

三人称視点では、登場人物全員の心情を書けます。

一人称視点では、全ての描写が主人公の視点となるため、主人公以外の登場人物の心情を主人公の予想でしか書けません。例えば、内心で怒っている気がする、という描写はできますが、内心で怒っている、と断言できません。

一方で、三人称小説では、全ての登場人物の心情も行動も客観的に描写できるため、読者だけがわかる登場人物の感情のすれ違いなども描写できます。

## 登場人物全員が知らないことを書ける

三人称視点では、登場人物全員が知らないことを書けます。

一人称視点では、登場人物全員が知らない情報を主人公が知ることはできず、主人公が知ることができない情報を描写することができません。

一方で、三人称小説では、天の声視点で登場人物全員が知らないことを描写できるため、読者にしかわからない未来の楽しみや恐怖を描写し、先の展開へのワクワクを増すことができます。

# まとめ

いかがだったでしょうか。
今回の記事では、人称視点とその特徴について説明しました。
小説を書き始める時に少しでも役立てば幸いです。

また、一人称小説、二人称小説、三人称小説は、一長一短があります。
自分はどちらが書きやすいか、自分が書こうとしているジャンルの小説ではどちらが向いているかを考えながら、楽しい執筆ライフを送りましょう。
