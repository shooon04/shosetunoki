# 描写の種類

## 心理描写とは

心理描写とネットで調べてみると、
「文学作品などで、人物の心の微妙な移り変わりを細かく分析して描き出すこと」(スーパー大辞林)
と書かれています。

難しく書かれていますね。簡単に言いますと、
「主人公の気持ちを文章にする」
です。

難しいと思われがちですが、意外と簡単です！


## 行動描写とは

行動描写は人物の行動を書くものです。僕は心理描写の次に必要な要素だと思っています。

主人公の行動、小説に出てくるほかのキャラの小さな行動。
それを書くのは心理描写よりは簡単ではありませんか？

## 情景描写とは

情景描写とは、情景と書いてあるように、その場の風景を書きます。
情景描写は心理描写、行動描写に多くの効果を与えることが多いです。

情景描写は心理描写、行動描写とは違い、入れるところにもポイントがあります。どこに入れるかについては後述の「描写を入れてみよう」で詳しく説明したいと思います。


# 描写ってなんで入れなきゃいけないの？

描写を入れるのには大きく分けて２つの理由があります。

## 「」（カギカッコ）の文章だけでは伝わらないものがある

会話だけの文章だと、主人公の心情が伝わらないことがあります。
例えば、
「今日楽しかった」
「よかった」
という文があるとします。

「よかった」という文がなかった場合を考えてください。

「今日楽しかった！」
だと、発言者は楽しんだんだなということがわかります。

しかし、読み方を変えて、

「今日楽しかった？」

にしてみるとどうでしょう。聞いているのでこの発言者が楽しんだかどうかが伝わりません。逆に発言者とは異なる人に聞いているようにも見えます。

最近の小説ではほとんどの場合、「！」や「？」をつけますが、一昔前までは「！」や「？」は使いませんでした。それがなくても伝わる文を、これを読んでいる皆さんには書いてもらいたいと思っています。
そこで必要なのが描写です。

例えば先程の会話文の間に以下のことが入っていたらどうでしょう。

「彼女は僕の目をまっすぐ見ながら言った。その目は笑っている。」

繋げてみると以下のようになります。

「今日は楽しかった」
彼女は僕の目をまっすぐ見ながら言った。その目は笑っている。
「よかった」

彼女が楽しんだことがわかります。描写は読者にわかりやすくするために、細かな心情を伝えるために必要なのです。

## 主人公の心情を引き立てる

例えば、落ち込んでいる人がいます。
それを文書にしようとした時に、

「彼はやる気を失ってしまった。」

だけだと、サクッと終わってしまいます。しかし、そこに行動描写を入れてみると……

「彼はやる気を失ってしまった。その場に崩れ、ピクリとも動かない。」

どうでしょう。より落ち込んでいるように見えませんか？
また、情景描写を入れてみると……

「彼はやる気を失ってしまった。沈みかけていた太陽が山に隠れ、暗闇にカラスの鳴き声だけが響いている。」

どうでしょうか。行動描写を入れた時とは異なる落ち込みが表現されていると思いませんか？
これが、描写を書くべき理由の２つ目です。


# 描写を入れてみよう！

## まずは心理描写と行動描写

では実際に描写を入れてみましょう。このタイトルにもありますようにまずは、心理描写と行動描写です。以下の会話文を見て下さい。

……①
A　「ねぇ、どうして。なんで浮気なんかしたの」
……②
B　「そ、それは……」
……③
A　「黙らないでなんか言いなよ。ごめんなさいも言えないの」
……④

これをみる限り、カップルで喧嘩をしている、その理由は浮気だとわかります。それではこの文に描写を挟んでいきましょう。

突然ですがここでクイズです。描写を入れる場所は上の①〜④の内、どこが良いと思いますか？　Bの視点で物語が進んでいるとします。
人によって入れたいと思う場所は異なると思いますが、どこに入れたら「」だけの文章から変わるでしょうか？

この文に僕が描写を入れるとしたら②です。
Aの人がBの人に対して問い詰めているシーンなので、Bの人から見たAの気迫や目力などを書くのが一番かなと思います。

では実際に入れていきます。僕が入れるとしたらこのような文を入れます。

「Aは床をじっと見つめながら小声で言った。その声は今まで聞いたことのないものだった。浮気をされた人は強いという言葉は合っているのかもしれない。」

どうでしょうか。Aの言葉がさらに強調されていると思いませんか？
繋げてみるともっとわかりやすいかもしれません。

「ねぇ、どうして。なんで浮気なんかしたの」

Aは床をじっと見つめながら小声で言った。その声は今まで聞いたことのないものだった。浮気をされた人は強いという言葉は合っているのかもしれない。

「そ、それは……」
「黙らないでなんか言いなよ。ごめんなさいも言えないの」

描写を入れるだけでここまで変わります。
では次に情景描写を入れていきましょう。

## 情景描写を入れよう！

情景描写についても上で使った会話文＋描写のものに加えていきます。

……①
「ねぇ、どうして。なんで浮気なんかしたの」

Aは床をじっと見つめながら小声で言った。その声は今まで聞いたことのないものだった。浮気をされた人は強いという言葉は合っているのかもしれない。

「そ、それは……」
……②
「黙らないでなんか言いなよ。ごめんなさいも言えないの」
……③

以上の文が先程作ったものですね。
これで十分かと思われますが、これに情景描写を入れるともっと文章に厚みが増します。

では上の①〜③のうちどこに情景描写を入れるといいでしょうか？

僕だったら③に入れます。
①は会話文のスタートなので入れないです。逆に②は、「それは」の後に「……」が入っているので、ここに描写を入れると間が悪くなってしまいます。

ではどのような情景描写を入れるのか。
想像してみてください。
この話はどこで起きていると思いますか？

これについては本当に人によって分かれると思います。
公園かもしれない。カフェかもしれない。今回は家の中で会話をしていると想定します。

家の中で音を出すものと言ったらなんでしょう？
たくさんあると思います。
テレビや、空気清浄機、ケトル、コーヒーマシン。出していったらキリがないです。

では、実際に③に描写を入れていきましょう。

皆さんも考えた後、下を見てください。僕の考えた描写の例を２つ載せますので、こんな感じの描写もできるんだと思ってみてください。

例１
部屋の中は静寂に包まれていた。いつもはガタガタうるさい洗濯機も、夕飯の後には必ずと言っていいほど飲むコーヒーを入れるためのマシンも。音を立てず、まるで忍者のように存在を消している。僕の味方は誰もいないように見えた。

例２
僕の背中からテレビの音が聞こえた。話題は最悪なことに夫の浮気についてだ。消したい衝動に駆られたが、ここで消したらさらにAの怒りを買うだろう。僕は黙ることしかできなかった。

どうでしょうか？
例１も例２も浮気をしてしまったB視点の描写です。違いが出ているのがわかるでしょうか？


# まとめ

今回は描写についての記事でした。
この記事を書こうと思ったきっかけは、僕の友達がカギカッコだけの文になってしまうのが悩みだと言っていたのがきっかけでした。

小説を書くうえで、その人の「個性」が出るのが描写です。

描写を書けるようになると小説の厚みも変わります。

話の中に自分も入り込んだ気持ちになって描写を書いてみてください。
