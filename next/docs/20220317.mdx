# SS小説って？

## 超短編と呼ばれるSS小説

小説には、「長編小説」、「中編小説」、「短編小説」と種類があります。その中の「短編小説」よりも短い小説のことを「ショートショート」、「ショートショート」よりさらに短い小説のことを「掌編小説」と呼びます。「SS(ショートストーリー)小説」とは「ショートショート」と「掌編小説」をまとめた言い方となります。

## 「ショートショート」と「掌編小説」

ここまで読まれて、
「じゃあ、ショートショートと掌編小説の違いってどこで決まるんだ？」
と思われた方がいると思います。
「ショートショート」と「掌編小説」には大きく分けて2つの違いがあります。

1つ目は、「文字数の違い」です。

「ショートショート」は、一般的に400字詰め原稿用紙20枚(文字数に換算すると8000字)に満たない作品のことを言い、「掌編小説」は、300字から800字程度の作品のことを言います。「掌編小説」の方は、原稿用紙で表しません。

2つ目は、「自由さの違い」です。

「掌編小説」は上では300字からと言いましたが、正直なところ**1文字**だけでも作品になります。「あ」と打っただけでも作品になり得ます。しかし、「ショートショート」は「あ」だけで投稿したら「なんだこれ、馬鹿じゃないの？」と思われます。

ここで一旦まとめると、

- SS小説は「掌編小説」と「ショートショート」をまとめた言い方
- 「掌編小説」と「ショートショート」の違いは文字数と自由さ

になります。

## どこに投稿するの？

今まで述べてきたように、SS小説というのはとても短い文章です。なので、SS小説は「Twitter」に出すことが多いです。

Twitterはツイートをする時の文字制限があります。全角で140字、半角で280字です。

その文字制限の中で、小説を書きます。Twitterの文字数制限を利用したSS小説です。通称「Twitter小説」とも呼ばれます。

また他には、短編集として「カクヨム」や「エブリスタ」に投稿されることが多いです。「小説家になろう」はあまりお勧めしません。異世界転生のSS小説を書くならまだしも、恋愛小説や青春小説はあまり読まれないからです。

## どんなタグをつけるの？

多くの人に読まれるには、「タグ」をうまく使うことが重要です。Twitterに投稿するにせよ、カクヨムやエブリスタに投稿するにせよ、「タグ」というのは大きな力を持っています。今回は僕が実際に使っているタグをお教えしたいと思います。

タグは短い文字の中で「こんな作品だよ」と示すものです。なので、

- 短編
- SS小説
- 掌編小説

と書きます。

そのあとは自分のジャンルに合わせてタグを書いていきます。

# SS小説を書いてみよう！

## 少ない文字数でまとめるのは難しいっ！

SS小説について知ったところで、書いてみましょう！……と言いたいところですが、すぐに書けるというものではありません。慣れるまでには時間がかかります。

特に300字以内で書く掌編小説を書くのは難しい！

なぜかと言うと、少ない文字数制限の中で、読者を自分の物語の世界観に引き込まなきゃいけないからです。恋愛小説だったら読者をキュンとさせる、ホラー小説だったら読者に恐怖を与える。それを少ない文字数でやらなければいけません。じゃあ、どうするのか！

それは

「自分の書きたい物語に入れたい言葉を並べてみる」

です。

スマホの中に入っている「メモ機能」を使っても良し。Twitterのツイートするところを使っても良し。紙に書いても良し。繋げようとせず、箇条書きで書いてみましょう。そして文字数を数えてみましょう。この時点で140字(Twitterの場合)、300字を超えていたら、NGです。まあ、超えることはほとんどないと思いますが……。

それをしたあと、実際に書いていきます。

書き終わった後に、あと数文字削らなきゃというときがあると思います。そう言う時は、推奨したくはありませんが、「。」を消したり、「……」をひとつにしたり、と少し電子小説のルールから外れても構いません。

## 起承転結って守らなきゃいけない？

小説には「起承転結」がつきものと思っている方。

違います！

特に、SS小説の世界だと「起転結」が多いです！

短い文章に「承」を入れるのは難しいのです。そうなの？　と思う方は一度書いてみてください。「承」を書いている間に文字数制限が来ます。

なので、SS小説を書く上で、絶対に「起承転結」というスタイルを守らなきゃと身構えなくても大丈夫です。この点に関しては普通の小説より書きやすいかもしれません。

## 題名って必要？

SS小説を書く際に題名っているのかな？　と思われる方がいると思います。特に、Twitterに出す場合は題名をリプ欄につけるかつけないかで悩むと思います。

僕のことをお話しすると、僕は必ずつけています。理由はカクヨムさんに短編集として作品を載せているからです。しかし、もし、カクヨムさんに掲載していなかったら？　僕は題名をつけていないと思います。

題名というのは小説の中でも結構重要な役割を持っています。題名を読んで物語の全てが繋がることがあります。題名でこんな話なのかなと思わせることもできます。

逆に題名をつけなかったら？

読者の想像を広げることができます。

なので、題名は必ずつけなくても大丈夫です。自分の作品に合わせてつけるかつけないかを決めましょう。

# まとめ

今回はSS小説についてのお話と、書き方について書かせていただきました。簡潔におさらいをしましょう。

- SS小説は掌編小説とショートショートをまとめた言い方
- 掌編小説とショートショートの違いは文字数と自由さ！
- 書くときは、まず言葉を箇条書きで書く
- 題名の有無は自分で決められる

どうでしょうか？
SS小説を知らなかった方は新しい世界を知ることができたのではないでしょうか？
また、書こうと思っている方は書き方がわかり、難しそうという抵抗感が減ったのではないでしょうか？

SS小説を書いて損をすることはないです。興味があるのであれば、書いてみるといいでしょう！
